Modal



ExampleModal
Nested modal
Name type default description
backdrop boolean or the string 'static' 'static' Includes a modal-backdrop element. Alternatively, specify static for a backdrop which doesn't close the modal on click.
keyboard boolean true Closes the modal when escape key is pressed
onBeforeChange function undefined Function that gets called before every change of modal state. If this function returns false, action is prevented.
All option defaults are configurable in 'BsModalConfigService' and can be changed via bindings.
